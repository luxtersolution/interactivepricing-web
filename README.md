# interactivepricing-web
Web client for interactivepricing.com engine

1. ```bash
   git clone https://github.com/luxtersolution/interactivepricing-web.git
   ```
1. ```bash
   npm install
   ```
1. ```bash
   npm build
   ```
1. ```bash
   npm start
   ```
1. Open http://localhost:3000 in a browser
